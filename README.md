## Configurações iniciais para rodar o projeto

```
# clonar o projeto
git clone git@github.com:Makson19/tour_of_heroes_api.git

# instalar as dependências do Ruby on Rails
bundle install

# criar as tabelas
rails db:migrate
```

## Rodar o projeto

```
rails s
```
